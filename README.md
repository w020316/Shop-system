# 微商城 - WeChat Store

基于 Vue 3 + TypeScript + Pinia + Vue Router 的移动端电商应用

## 技术栈

- Vue 3.4 (Composition API + TypeScript)
- Vite 5 (构建工具)
- Pinia 3 (状态管理)
- Vue Router 4 (路由管理)
- Tailwind CSS 3 (原子化CSS)
- Lucide Vue Next (图标库)
- localStorage (数据持久化)

## 功能模块

- 商品浏览与搜索
- 分类浏览（左右双栏布局）
- 商品详情（图片轮播+规格选择+收藏）
- 购物车管理
- 用户注册与登录
- 订单系统（创建→支付→发货→确认收货）
- 收货地址管理
- 商品收藏
- 浏览历史记录
- 商品后台管理
- PC端侧边栏 + 移动端TabBar 响应式布局

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run check
```
